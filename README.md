# LegacyLens 2.0 ✨

> **Bridge the gap between English and SQL.**

LegacyLens is a production-grade, AI-powered Natural Language Database Interface that allows non-technical users to speak directly to complex relational (MySQL) and NoSQL (MongoDB) databases. 

By unifying edge-runtime serverless inference via Next.js and advanced Pollinations AI mapping, LegacyLens converts standard English questions into precise database executions instantly.

![LegacyLens Hero](https://images.unsplash.com/photo-1551288049-bebda4e38f71?q=80&w=800&auto=format&fit=crop)

## Features

- **Multi-Database Agility**: Dynamic runtime schema sniffing for both `MySQL` and `MongoDB`.
- **Advanced Natural Language Inference**: Converts complex analytical English questions into robust query dialects utilizing integrated AI processing.
- **Serverless Edge Engine**: Built securely strictly on Next.js `14` App Router architecture with secure isolated API gateways protecting raw credentials.
- **Universal Accessibility Suite**: Inclusive natively. Hardcoded dyselxia-friendly rendering states and dynamic `font-size` scalers prioritizing A11y out of the box.
- **Hardware Voice Recognition**: Bypass typing entirely; natively hooks into WebKit `SpeechRecognition` engines with continuous context tracking.
- **Micro-Physics UI**: Breathtaking dark-cyan aesthetics powered by `framer-motion` scroll-parallax and interactive component elasticity.

## 🚀 Deployment

This application is officially deployed as a stateless architecture via **Vercel**.

## 🛠 Tech Stack

- **Framework**: [Next.js 14 App Router](https://nextjs.org/)
- **UI & Animations**: React 18, Framer Motion, Vanilla CSS Modules
- **AI Core**: Pollinations AI API / Gemini 2.5 Inference Fallbacks
- **Drivers**: `mysql2`, `mongodb`
